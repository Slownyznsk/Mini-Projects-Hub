# Mini-Projects-Hub
Pequeno hub com 3 projetos para completar do JSDescomplicado, concluídos e estilizados por mim.

Essa atividade teve como objetivo explorar mais profundamente o JavaScript e suas possibilidades de integração com HTML e CSS. Para isso, utilizamos o site jsdecomplicado.online como referência para compreender melhor a linguagem. Com base no conteúdo estudado e nos projetos disponibilizados no site, desenvolvemos nossas próprias versões, aplicando o que aprendemos na prática.
